# Youtube-game

🎮 Community-Driven Chaos Platformer

This is a fast-paced 2D platformer built entirely around community ideas, chaotic mechanics, and evolving gameplay. What started as a simple project has grown into a feature-packed game where almost everything can (and probably will) change.

🚀 Core Concept

Players explore a large, tile-based world filled with enemies, traps, and strange mechanics—many of which were inspired directly by community suggestions. The game blends classic platforming with unpredictable systems, making every playthrough feel unique.

🧠 Features

🧀 Dynamic Gameplay Systems
From collectible cheese to bizarre mechanics like potato cannons, the game constantly introduces new ideas.

🥔 Potato Cannon Combat
Aim and fire projectiles with physics, upgrades, and explosive variants.

👾 Variety of Enemies
Snails, bats, turrets, snowmen, and more—each with different behaviors.

🌍 Large Scrollable World
Built with a tile-based system and multiple levels/maps.

🎨 Pixel Art + Visual Effects
Includes lighting, shadows, particles, CRT effects, and optional pixel scaling.

⚙️ Extensive Settings System
Customize graphics, controls, accessibility, and gameplay modifiers.

🎮 Multi-Input Support
Keyboard, mouse, touch controls, and full gamepad support.

📈 Upgrade System
Level up and choose from randomized upgrades that change how you play.

🧪 Experimental Mechanics
Expect weird, funny, and sometimes broken features—this game embraces chaos.

🛠️ Built With

HTML5 Canvas

JavaScript (vanilla)

Custom game loop + physics system

🌱 Ongoing Development

This game is actively evolving, with new features, mechanics, and content being added regularly—often based on community feedback.

💡 Inspiration

This project is heavily inspired by the idea of building a game from player comments, turning suggestions into real, playable features.
